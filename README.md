# SecurityAgentPlugin

A collection of skills, plugins, and tools designed to enhance vulnerability detection and security
scanning capabilities for AI agents.

## Project Structure

```text
SecurityAgentPlugin/
  Skills/       # Agent skills for security workflows
  Plugins/      # Agent plugins for extended functionality
  Tools/        # Standalone security utilities
```

## Skills

- **security-scan** - General multi-agent vulnerability scanning with structured discovery,
  verification, and validation for codebases.
- **aiml-interface-boundary-scan** - AI/ML third-party component interface-boundary scanning for GPU
  compute platforms, focused on exposed interfaces, trust-boundary mismatches, and candidate
  vulnerability verification planning.

## Status

Under active development. More skills, plugins, and tools coming soon.
