# SecurityAgentPlugin

A collection of skills, plugins, and tools designed to enhance the vulnerability detection and security scanning capabilities of AI agents.

## Project Structure

```
SecurityAgentPlugin/
├── Skills/       # Agent skills for security workflows
├── Plugins/      # Agent plugins for extended functionality
└── Tools/        # Standalone security utilities
```

## Skills

- **security-scan** — Multi-agent vulnerability scanning with structured discovery, verification, and validation phases.

## Status

Under active development. More skills, plugins, and tools coming soon.
